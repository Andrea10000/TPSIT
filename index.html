JAVA

Java è un linguaggio compilato in cui hanno introdotto il concetto di macchina virtuale (dentro Windows), visto che i file compilati sono diversi per ogni sistema operativo, ma con la macchina virtuale ci può essere un file compilato universale (sulla macchina virtuale), però questo è tutto più lento e le macchine virtuali possono avere differenze. Infatti la stessa macchina virtuale istallata su più computer con sistema operativi diversi, può avere differenze.
