Tabella delle formattazioni (titolo h2)

tag = descrizione
 <strong>   =  Definisce un testo importante 
 <b>   =  Definisce un testo in grassetto 
 <em>   =  Definisce un testo con enfasi 
 <i>   =  Definisce un testo 'italico', in corsivo 
 <small>   =  Testo normale Definisce un testo in un formato più piccolo 
 <sub>   =  Testo normale Definisce un testo in posizione pedice 
 <sup>   =  Testo normale Definisce un testo in posizione apice 
 <ins>   =  Definisce un testo sottolineato 
 <del>   =  Definisce un testo sbarrato 
 <mark>   = Definisce un testo evidenziato
